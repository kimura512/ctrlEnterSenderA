# i18n Updates (v1.3.2)

## 概要

v1.2.0 以降に追加・変更されたメッセージ（主にオンボーディングとオプションページ関連）を、全37言語のロジックファイルに反映しました。

## 対応内容

- **基準**: `_locales/en/messages.json` を最新の日本語版（`ja`）に合わせて更新。
- **反映**: 更新された英語メッセージを、他の全言語（`ar`〜`zh_TW`）に適用（Fallbackとして英語を使用）。

## 更新されたキー

以下のキーが全言語で最新の英語記述に統一されました：

1. `onboardingDescription` (Enterキーの挙動説明)
2. `onboardingSiteToggle` (ポップアップでの切り替え説明)
3. `onboardingDefaultEnabled` (デフォルト設定の説明)
4. `onboardingAdvancedSettings` (詳細設定の説明)
5. `optionsDescription` (オプション冒頭の説明)

## 検証

- `scripts/update-i18n.js` により全34フォルダが更新されたことを確認。
- ビルド (`pnpm build`) が正常に完了し、JSON構文エラーがないことを確認。
- ランダムサンプリング (`fr/messages.json`) で更新反映を確認。
