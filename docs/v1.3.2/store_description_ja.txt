私の個人的な統計によると、、、

● 新入社員の5人に1人は、SlackやTeamsで書きかけメッセージを誤送信し、社会の洗礼を受けます。
● そして ChatGPT では、未完成の文章を送ってしまい、勝手に回答が始まって絶望する人が「5人中5人」（つまり全人類）。

さらに、こんな“事故”も日常茶飯事です：

【事例1】
@everyone
（メンションだけ送信 → こいつミスったなと全員にバレる恥辱。）

【事例2】
小林
（上司に「小林さん、お疲れ様です」と打つはずが、名前だけで発射。こわいよ。）

こんな冷やっとする誤送信は、もう今日で終わりにしませんか？

◆ 主な機能
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

▼ 誤送信を防止
Enterキーでメッセージが送信されることはもうありません。
ただ改行が挿入されます。そう、文明的なコミュニケーションのあるべき姿です。

▼ 統一されたショートカット
対応するほぼすべてのサイトで、Ctrl+Enter（MacではCmd+Enter）で送信できます。

▼ 選べる起動モード
「デフォルトON（ブラックリスト）」と「デフォルトOFF（ホワイトリスト）」モードを選択できるようになりました。
アイコンからサイト単位でのON/OFF切り替えも可能。

▼ 主要プラットフォームに対応
AI: ChatGPT / Claude / Gemini / Grok / DeepSeek / Perplexity / Z.ai
仕事: Slack / Microsoft Teams / Google Meet
SNS: Discord / Facebook Messenger / X (Twitter) / WeChat / Telegram
…その他多数。
※Google Chat および YouTube Live チャットは現在未対応です。
