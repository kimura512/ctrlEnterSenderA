▼ منع الإرسال العرضي
مفتاح Enter لم يعد يرسل الرسائل.
بل يقوم ببساطة بإدراج سطر جديد — تماماً كما يجب أن يكون التواصل المتحضر.

▼ اختصارات موحدة في كل مكان
أرسل باستخدام Ctrl+Enter (أو Cmd+Enter على Mac) عبر جميع المواقع المدعومة.

▼ أوضاع التفعيل
يمكنك الآن الاختيار بين وضعي "التفعيل افتراضياً (القائمة السوداء)" و"الإيقاف افتراضياً (القائمة البيضاء)" عبر الإعدادات.
قم بتبديل التفعيل/الإيقاف لكل موقع على حدة عبر الأيقونة.

▼ يعمل على المنصات الرئيسية
الذكاء الاصطناعي: ChatGPT / Claude / Gemini / Grok / DeepSeek / Perplexity / Z.ai
العمل: Slack / Microsoft Teams / Google Meet
التواصل الاجتماعي: Discord / Facebook Messenger / X (Twitter) / WeChat / Telegram
…والعديد من المواقع الأخرى.
*ملاحظة: Google Chat و YouTube Live Chat غير مدعومين حالياً.*
