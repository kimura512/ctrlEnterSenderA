◆ 主要功能
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

▼ 防止誤發
按 Enter 鍵不再發送消息。
它只會插入換行符——這才是文明溝通該有的樣子。

▼ 到處通用的統一快捷鍵
在所有支持的網站上通過 Ctrl+Enter（Mac 上為 Cmd+Enter）發送。

▼ 激活模式
現在您可以在設置中選擇“默認開啟（黑名單）”和“默認關閉（白名單）”模式。
通過圖標針對每個網站單獨開啟/關閉擴充功能。

▼ 支持主要平台
AI: ChatGPT / Claude / Gemini / Grok / DeepSeek / Perplexity / Z.ai
辦公: Slack / Microsoft Teams / Google Meet
社交: Discord / Facebook Messenger / X (Twitter) / WeChat / Telegram
……以及更多。
*注意：目前不支持 Google Chat 和 YouTube 直播聊天。*
