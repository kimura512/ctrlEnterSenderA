# Ctrl+Enter Sender

様々なチャットアプリケーションやウェブサイトで、`Ctrl+Enter`でメッセージを送信し、`Enter`で改行を挿入できるようにするChrome拡張機能です。

## 機能

- **Ctrl+Enterで送信**: `Ctrl+Enter`（Macでは`Cmd+Enter`）を押すと、メッセージが即座に送信されます。
- **Enterで改行**: `Enter`キーのデフォルトの送信動作を防ぎ、代わりに改行を挿入します。
- **幅広い互換性**: 以下のような人気プラットフォームで、すぐに動作します：
  - Slack
  - Discord
  - Microsoft Teams
  - Google Chat / Meet
  - Twitter (X)
  - Facebook Messenger
  - その他多数！
- **スマート検出**: 複数行テキスト入力欄やcontent-editableエリアを自動検出します。
- **カスタマイズ可能**:
  - ドメインごとに有効/無効を切り替え可能
  - 一部のサイト（X/Twitter、Google検索など）はデフォルトで無効

## インストール

### ソースから

1. このリポジトリをクローンします。
2. 依存関係をインストールします：
   ```bash
   pnpm install
   ```
3. 拡張機能をビルドします：
   ```bash
   pnpm run build
   ```
4. Chromeを開き、`chrome://extensions/`に移動します。
5. 右上の「デベロッパーモード」を有効にします。
6. 「パッケージ化されていない拡張機能を読み込む」をクリックし、プロジェクトディレクトリ内に生成された`dist`フォルダを選択します。

## 使用方法

インストール後、拡張機能は対応サイトで自動的に動作します。

- **Ctrl+Enter (MacではCmd+Enter)**: メッセージを送信
- **Enter**: 改行を挿入

### ポップアップメニュー

ツールバーの拡張機能アイコンをクリックすると、現在のサイトの設定にアクセスできます：

- **有効/無効**: 現在のドメインで拡張機能を有効/無効に切り替え

### オプションページ

拡張機能アイコンを右クリックして「オプション」を選択すると、すべてのドメインの設定を表示・管理できます。

- **初期設定済みドメイン**: デフォルトで無効になっているドメイン（X/Twitter、Google検索など）
- **ユーザー設定ドメイン**: ユーザーが設定したドメイン
- **初期化ボタン**: すべての設定をインストール時の状態に戻す

- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [CRXJS Vite Plugin](https://crxjs.dev/vite-plugin)

### コマンド

- `pnpm run dev`: 開発サーバーを起動（HMR対応）
- `pnpm run build`: 本番用に拡張機能をビルド

## 多言語対応

この拡張機能は30以上の言語に対応しています。ブラウザの言語設定に応じて自動的に適切な言語が選択されます。
