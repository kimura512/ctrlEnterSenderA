▼ Cegah Pengiriman Tidak Disengaja
Enter tidak lagi mengirim pesan.
Itu hanya menyisipkan baris baru—persis bagaimana komunikasi yang beradab seharusnya.

▼ Pintasan Terpadu Di Mana Saja
Kirim dengan Ctrl+Enter (atau Cmd+Enter di Mac) di semua situs yang didukung.

▼ Bekerja di Platform Utama
AI: ChatGPT / Claude / Gemini
Kerja: Slack / Microsoft Teams / Google Meet
Sosial: Discord / Facebook Messenger / X (Twitter)
...dan banyak lagi.
*Catatan: Google Chat dan YouTube Live Chat saat ini tidak didukung.*

▼ Deteksi & Kontrol Cerdas
Secara otomatis mendeteksi input obrolan di sebagian besar situs web.
Aktifkan/Nonaktifkan ekstensi per situs melalui ikon.
Sekarang lebih sederhana dari sebelumnya—tidak ada mode yang rumit, hanya berfungsi.
