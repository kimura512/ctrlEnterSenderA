★★ 别再给老板误发“情书”了。 ★★

这里有一些虽然是虚构的、但听起来在情感上非常真实的统计数据：

● 1/5 的新员工通过发送未完成的 Slack/Teams 消息在工作中接受“洗礼”。

然后还有这些……真正恐怖的事故：

【示例 1】
“挑衅性拼写错误” 你输入的是："I will kill you"（原意："I will kill your ticket in the system." 我会在系统中关闭你的工单。）

【示例 2】
“猝不及防的表白” 你输入的是："I love u"（原意："I love updates like this one." 我喜欢这样的更新。）

我们都经历过这些。但我们不必一直忍受。

◆ 主要功能
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

▼ 防止误发
按 Enter 键不再发送消息。
它只会插入换行符——这才是文明沟通该有的样子。

▼ 到处通用的统一快捷键
在所有支持的网站上通过 Ctrl+Enter（Mac 上为 Cmd+Enter）发送。

▼ 激活模式
现在您可以在设置中选择“默认开启（黑名单）”和“默认关闭（白名单）”模式。
通过图标针对每个网站单独开启/关闭扩展程序。

▼ 支持主要平台
AI: ChatGPT / Claude / Gemini / Grok / DeepSeek / Perplexity / Z.ai
办公: Slack / Microsoft Teams / Google Meet
社交: Discord / Facebook Messenger / X (Twitter) / WeChat / Telegram
……以及更多。
*注意：目前不支持 Google Chat 和 YouTube 直播聊天。*
