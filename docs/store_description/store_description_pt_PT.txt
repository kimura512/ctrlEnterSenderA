◆ Principais Recursos

▼ Prevenir envios acidentais
O Enter já não envia mensagens.
Simplesmente insere uma nova linha — exactamente como uma comunicação civilizada deve ser.

▼ Atalhos unificados em todos os lugares
Envie com Ctrl+Enter (ou Cmd+Enter no Mac) em todos os sites suportados.

▼ Modos de Activação
Agora pode escolher entre os modos "Activado por padrão (Lista negra)" e "Desactivado por padrão (Lista branca)" via opções.
Active/desactive a extensão por site através do ícone.

Trabalho: Slack / Microsoft Teams / Google Meet
Social: Discord / Facebook Messenger / X (Twitter) / WeChat / Telegram
