◆ 主要功能
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

▼ 防止誤發送
Enter鍵不再發送訊息。
它只會插入換行符——這才是文明溝通應有的方式。

▼ 統一快捷鍵
在所有支援的網站上，使用 Ctrl+Enter（Mac 上為 Cmd+Enter）發送訊息。

▼ 支援主要平台
AI：ChatGPT / Claude / Gemini
工作：Slack / Microsoft Teams / Google Meet
社交：Discord / Facebook Messenger / X (Twitter)
……以及更多。
*注意：目前不支援 Google Chat 和 YouTube Live Chat。*

▼ 智能檢測與控制
自動檢測大多數網站上的聊天輸入。
透過圖示切換每個網站的擴充功能開/關。
現在比以往更簡單——沒有複雜的模式，直接可用。
